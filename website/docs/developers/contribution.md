# Contribution Guide

## Code of Conduct
All contributors must follow the [Code of Conduct](https://github.com/nova-ecosystem/.github/blob/main/CODE_OF_CONDUCT.md).

## How to Contribute
1. Fork the repository.
2. Create a feature branch.
3. Make changes and commit with clear messages.
4. Open a Pull Request.

## Issue Reporting
Use GitHub Issues with templates for bug reports and feature requests.

## Review Process
- Automated CI checks must pass.
- Maintainers review and approve.
- Merge strategy: squash commits for clarity.
