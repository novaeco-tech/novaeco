# FAQ

Frequently asked questions about the Nova Ecosystem.

---

## General
**Q: What is the Nova Ecosystem?**  
A: A collaborative infrastructure for circular economy, built on enablers, sectors, workers, and products.

**Q: How do I join as a contributor?**  
A: Register via the Identity service (`/auth`) and follow the [Contribution Guide](./developers/contribution.md).

---

## Development
**Q: How do I run the ecosystem locally?**  
A: Use DevContainers. See [Quickstart](./developers/quickstart.md).

**Q: How do I add a new worker?**  
A: Define it in the monorepo, document it in `docs/usecases`, and link it to its enabler/sector.

---

## Governance
**Q: How are decisions made?**  
A: Through ADRs and NovaPolicy governance modules.

**Q: How do I propose an ADR?**  
A: Create a new file in `docs/architecture/` with the format `adr-XXX-title.md`.

---

## Documentation
**Q: How do I contribute to docs?**  
A: Fork the repo, add or edit `.md` files, and update `sidebars.js`.
