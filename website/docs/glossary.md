# Glossary

This glossary defines common terms used across the Nova Ecosystem.

## Core Concepts
- **Enabler** — Horizontal service that provides foundational capabilities (e.g., Identity, NovaHub, NovaFinance).
- **Sector** — Application industry where circular solutions are deployed (e.g., NovaAgro, NovaBuild).
- **Worker** — Microservice that performs a specialized task (e.g., NovaMaterial — DPP sync).
- **Product** — Hardware or software innovation built on the ecosystem (e.g., SortBot X1, NovaBox).

## Technical Terms
- **ADR (Architecture Decision Record)** — Documented decision about system design.
- **DPP (Digital Product Passport)** — Standardized digital record of a product’s materials and lifecycle.
- **LCA (Life Cycle Assessment)** — Methodology for assessing environmental impacts across a product’s lifecycle.
- **IAM (Identity and Access Management)** — Framework for managing user identities and permissions.
- **CI/CD (Continuous Integration/Continuous Deployment)** — Automated workflows for testing and releasing software.

## Governance Terms
- **Contribution Guide** — Rules and steps for contributing to the ecosystem.
- **Code of Conduct** — Standards of behavior for contributors.
- **Policy-as-Code** — Encoding governance rules into automated checks.

## Sustainability Terms
- **Circular Economy** — Economic system aimed at eliminating waste and reusing resources.
- **Urban Mining** — Recovering valuable materials from existing infrastructure and waste streams.
- **Product-as-a-Service** — Business model where products are leased or shared rather than sold.
