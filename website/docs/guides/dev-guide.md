# Developer Guide

This guide provides practical steps for working with the Nova Ecosystem Core.

## Local Development
- Use DevContainers for consistent environments.
- Attach to specific service containers (App, API, Auth, Website) via VS Code Remote Explorer.

## Adding Dependencies
- Python services: `pip install <package>`
- Website (Node.js): `npm install <package>`

## Debugging
- Switch container to "sleep" mode for manual debugging.
- Attach debugger in VS Code (F5).
